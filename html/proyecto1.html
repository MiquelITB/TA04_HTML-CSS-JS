Desarrollo de una aplicación móvil para la gestión de salud

  • Este proyecto se centra en el desarrollo de una aplicación móvil que permite a los usuarios llevar un control detallado de su salud
    mediante el registro de datos como el consumo de alimentos, actividad física, niveles de estrés y calidad del sueño.

  • Utilizando algoritmos de análisis de datos, la aplicación proporciona recomendaciones personalizadas para mejorar el bienestar general.

  • Este proyecto hace uso de tecnologías como la programación en Java y Kotlin para Android, así como el uso de bases de datos en la nube
    para el almacenamiento seguro de los datos del usuario.
